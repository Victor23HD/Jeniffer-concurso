# Matemática 06 — Raciocínio lógico

## Sequências numéricas e figurais

Ache a **regularidade**:
- +k, −k, ×k, ÷k
- alternância de duas regras
- potências / primos / dígitos

Ex.: 2, 6, 12, 20, 30, … → diferenças +4,+6,+8,+10 → próximo **42**

## Estruturas lógicas (proposições)

- **Negação** de “Todo A é B” → “Algum A não é B”  
- Negação de “Algum A é B” → “Nenhum A é B”  
- Negação de “Nenhum A é B” → “Algum A é B”

Conectivos:
- **e** (conjunção): verdade só se ambos verdadeiros  
- **ou** (disjunção): falso só se ambos falsos  
- **se… então** (condicional): falso **somente** quando V → F  
- **se e somente se** (bicondicional): mesmos valores

## Diagramas lógicos (conjuntos)

Use círculos (Venn) para “todos”, “alguns”, “nenhum”.  
Cuidado: “alguns” **não** impede que sejam todos — mas em provas, “alguns” = pelo menos um.

## Argumentação

- Premissas verdadeiras + forma válida → conclusão válida  
- Afirmar o consequente é **falácia**:  
  Se chove, a rua molha. A rua está molha. Logo chove? **Não necessariamente.**

## Relações entre pessoas / lugares

Monte tabela ou desenhe setas.  
Ex.: A é mais alto que B; C é mais baixo que B → A > B > C

## Checklist

- [ ] Sei negar “todo / algum / nenhum”
- [ ] Condicional só é falsa em V→F
- [ ] Em sequência, testo a regra no próximo termo
