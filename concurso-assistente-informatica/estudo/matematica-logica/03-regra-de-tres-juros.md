# Matemática 03 — Regra de três, juros e médias

## Regra de três simples

Monte a tabela e veja se é direta ou inversa.

**Direta:**  
`a está para b assim como c está para x` → `a/b = c/x`

Ex.: 3 impressoras fazem 90 páginas. 5 impressoras (mesmo ritmo) fazem?  
3 → 90 · 5 → x · direta → `3/5 = 90/x` → x = **150**

**Inversa:** multiplica em cruz “ao contrário” (ou inverte uma coluna).  
Ex.: 4 técnicos em 12 dias. Quantos dias com 6 técnicos?  
Mais gente → menos dias → `4×12 = 6×x` → x = **8**

## Regra de três composta

Mais de duas grandezas. Classifique cada uma (direta/inversa) em relação à que contém o “x”.

## Juro simples

`J = C · i · t`  
`M = C + J = C(1 + i·t)`

- C = capital  
- i = taxa **no mesmo período** de t  
- t = tempo  
- M = montante  

Ex.: R$ 2000 a 2% a.m. por 5 meses → J = 2000×0,02×5 = **200** · M = **2200**

Cuidado: 24% ao ano = 2% ao mês **só** em juro simples (dividir por 12).

## Médias

**Simples:** soma ÷ quantidade  
`(6+8+10)/3 = 8`

**Ponderada:** pesos  
`(nota1·p1 + nota2·p2)/(p1+p2)`  
Ex.: provas 7 (peso 2) e 9 (peso 3) → `(14+27)/5 = 8,2`
