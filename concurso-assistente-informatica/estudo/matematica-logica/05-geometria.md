# Matemática 05 — Geometria

## Ângulos

- Volta completa: 360°  
- Ângulo reto: 90° · raso: 180°  
- Complementares: somam 90° · suplementares: 180°

## Perímetro

Soma dos lados.  
Retângulo: `2(L + C)` · Quadrado: `4L` · Circunferência: `2πr` ou `πd`

## Áreas

| Figura | Área |
|---|---|
| Quadrado | L² |
| Retângulo | L·C |
| Triângulo | (b·h)/2 |
| Losango | (D·d)/2 |
| Trapézio | ((B+b)·h)/2 |
| Círculo | πr² |

π ≈ 3,14 (ou 22/7 se a banca indicar)

## Volumes / capacidade

| Sólido | Volume |
|---|---|
| Cubo | L³ |
| Paralelepípedo | L·C·A |
| Cilindro | πr²h |
| Prisma | área da base × altura |

## Teorema de Pitágoras

Triângulo **retângulo**: `a² + b² = c²` (c = hipotenusa, lado oposto ao ângulo de 90°)

Trios pitagóricos clássicos: 3-4-5 · 5-12-13 · 6-8-10 · 7-24-25

Ex.: catetos 6 e 8 → hipotenusa 10.
