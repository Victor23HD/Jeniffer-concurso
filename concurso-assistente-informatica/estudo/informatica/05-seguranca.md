# Informática 05 — Segurança da informação

## Princípios (CIA)

- **Confidencialidade:** só quem pode vê  
- **Integridade:** não foi alterado indevidamente  
- **Disponibilidade:** está acessível quando preciso  

## Ameaças comuns

| Ameaça | O que é |
|---|---|
| **Vírus** | código malicioso que se espalha (em geral precisa de “hospedeiro”) |
| **Worm** | espalha-se pela rede com menos dependência de arquivo |
| **Trojan** | disfarçado de programa legítimo |
| **Malware** | termo genérico para software malicioso |
| **Ransomware** | criptografa dados e pede resgate |
| **Spyware** | espiona |
| **Phishing** | golpe para roubar senha/dados (e-mail, SMS, site falso) |
| **Engenharia social** | manipular a pessoa (não só a máquina) |

## Senhas e autenticação

- Longas, únicas, não reutilizar  
- Gerenciador de senhas  
- **MFA / 2FA** (segundo fator: SMS, app, token)  
- Não anotar em post-it óbvio; não compartilhar  

## Boas práticas

- Manter SO e antivírus atualizados  
- Não clicar em links suspeitos  
- Conferir remetente e URL  
- Backup (também é segurança!)  
- Princípio do menor privilégio (usuário sem admin no dia a dia)  
- Bloquear a estação ao sair (Win+L)  
- Cuidado com pendrives desconhecidos  

## Phishing — sinais

- Urgência (“sua conta será bloqueada hoje”)  
- Erros de português / domínio estranho  
- Pedido de senha/dados por e-mail  
- Link que não bate com o site oficial  

## Checklist

- [ ] Malware é o “guarda-chuva”; vírus é um tipo
- [ ] Phishing ataca a pessoa
- [ ] Atualização + backup + senha forte + MFA
