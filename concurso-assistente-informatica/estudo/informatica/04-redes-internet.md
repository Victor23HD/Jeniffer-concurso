# Informática 04 — Redes, internet e e-mail

## Conceitos básicos de rede

| Termo | Significado |
|---|---|
| Rede | computadores/dispositivos conectados para compartilhar dados/recursos |
| LAN | rede local (empresa, prefeitura, casa) |
| WAN | rede ampla (interliga LANs; a internet é a maior) |
| Intranet | rede interna da organização (acesso restrito) |
| Internet | rede pública mundial |
| IP | endereço do dispositivo na rede |
| Máscara / gateway | definem rede local e “porta de saída” |
| DNS | traduz nome (www…) em IP |
| DHCP | entrega IP automaticamente |
| Switch | conecta dispositivos na LAN |
| Roteador | interliga redes / acesso à internet |
| Wi-Fi | rede sem fio |

Compartilhamento: pastas, impressoras, arquivos em rede (respeitando permissões).

## Configuração básica

- IP automático (DHCP) × IP fixo  
- Testes: `ping`, verificar cabo, Wi-Fi, proxy  
- Problema típico: “sem internet” → cabo? Wi-Fi? IP? DNS? gateway? site específico?

## Navegadores e web

- URL: endereço (https://…)  
- Link: atalho clicável para outra página/recurso  
- HTTP × **HTTPS** (HTTPS criptografa — cadeado)  
- Busca, favoritos, histórico, download, impressão de página  
- Cache/cookies: dados locais; limpar pode resolver páginas “estranhas”

## Correio eletrônico

- Conta, caixa de entrada, enviados, spam  
- **Para / Cc / Cco** (Cco esconde destinatários)  
- Anexos: cuidado com tamanho e tipo (.exe perigoso)  
- Assinatura, resposta, Encaminhar  
- Webmail × cliente (Outlook, Thunderbird)

Boas práticas: assunto claro, revisar destinatário, não abrir anexo suspeito.

## Checklist

- [ ] Intranet ≠ internet
- [ ] DNS resolve nomes; DHCP entrega IP
- [ ] Cco para listas sem expor e-mails
- [ ] HTTPS = conexão mais segura
