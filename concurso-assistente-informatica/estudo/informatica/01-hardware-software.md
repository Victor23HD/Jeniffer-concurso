# Informática 01 — Conceitos, hardware e software

## Hardware × Software

| | O que é | Exemplos |
|---|---|---|
| **Hardware** | parte física | CPU, RAM, HD/SSD, placa-mãe, monitor, teclado |
| **Software** | programas / instruções | Windows, Linux, Word, antivírus, drivers |

**Firmware:** software embutido em hardware (BIOS/UEFI).

## Componentes principais

| Peça | Função |
|---|---|
| **CPU (processador)** | “cérebro” — executa instruções |
| **RAM** | memória **volátil** de trabalho (apaga ao desligar) |
| **HD / SSD** | armazenamento **persistente** |
| **Placa-mãe** | conecta tudo |
| **Fonte** | energia |
| **GPU** | vídeo/gráficos |
| **Periféricos** | entrada (teclado, mouse, scanner) / saída (monitor, impressora) / mista (pendrive) |

### Bits e bytes (ordem de grandeza)

bit → Byte (8 bits) → KB → MB → GB → TB (×1024 aproximadamente em informática)

## Tipos de software

- **Sistema operacional (SO):** Windows, Linux, macOS  
- **Aplicativos:** Word, Chrome, Excel  
- **Utilitários:** compactadores, antivírus, backup  
- **Drivers:** permitem o SO falar com o dispositivo

## Ligação básica do PC

1. POST/BIOS ou UEFI verifica hardware  
2. Carrega o SO do disco  
3. Usuário inicia aplicativos (que usam RAM)

## Erros comuns cobrados

- “Salvei na RAM” → **errado** (RAM não guarda após desligar; salve em HD/SSD/nuvem)  
- Mais RAM ≠ mais “espaço de arquivos”; melhora desempenho com vários programas  
- SSD em geral é mais rápido que HD mecânico

- [ ] Distingo RAM × armazenamento
- [ ] Sei o papel de CPU, placa-mãe, driver
- [ ] Hardware físico × software lógico
