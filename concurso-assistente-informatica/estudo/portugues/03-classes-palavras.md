# Português 03 — Classes de palavras

## As 10 classes (resumo útil para prova)

| Classe | Função rápida | Exemplo |
|---|---|---|
| Substantivo | nomeia | computador, rede |
| Artigo | define o substantivo | o, a, um, uma |
| Adjetivo | caracteriza | rápido, seguro |
| Numeral | quantidade/ordem | dois, primeiro |
| Pronome | substitui/acompanha nome | ela, este, que |
| Verbo | ação/estado/fenômeno | instalar, ser |
| Advérbio | modifica verbo/adj/adv | bem, ontem, muito |
| Preposição | liga termos | de, em, para, com |
| Conjunção | liga orações/termos | e, mas, porque, se |
| Interjeição | emoção | puxa!, olá! |

## O que mais cai

### Pronomes relativos
- **que**, **quem**, **onde**, **cujo(a)**
- **cujo** concorda com o **possuído**, não com o possuidor  
  *O servidor **cujo** IP mudou…* / *A pasta **cujas** permissões…*

### Pronomes demonstrativos
- este/esta/isto (perto de quem fala / a seguir)
- esse/essa/isso (perto de quem ouve / anterior)
- aquele/aquela/aquilo (longe / mais remoto no texto)

### Verbos — tempos frequentes
- Presente, pretérito perfeito/imperfeito, futuro do presente
- Locução verbal: *vai instalar*, *tem estudado*

### Advérbio × adjetivo
- *Ela fala **rápido*** (advérbio — modo)
- *Um teste **rápido*** (adjetivo — qualifica substantivo)

### Conjunções (conectivos)
- Adição: e, nem
- Oposição: mas, porém, contudo
- Causa: porque, pois, já que
- Consequência: portanto, logo
- Condição: se, caso
- Concessão: embora, mesmo que
- Tempo: quando, enquanto
- Finalidade: para que, a fim de que

## Dica de prova

Se perguntarem “a palavra X exerce função de…”, veja o **papel na frase**, não só a aparência.  
*“O **que** você fez?”* → pronome interrogativo  
*“O arquivo **que** sumiu…”* → pronome relativo
