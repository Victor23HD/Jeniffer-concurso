# Português 04 — Concordância verbal e nominal

## Concordância verbal (verbo com sujeito)

### Regra geral
O verbo concorda com o **núcleo do sujeito**.

*Os computadores **estão** lentos.*  
*A instalação dos programas **demorou**.* (núcleo: instalação)

### Casos clássicos

| Caso | Como fica |
|---|---|
| Sujeito composto (antes do verbo) | plural: *O mouse e o teclado **funcionam**.* |
| Suj. composto depois do verbo | plural **ou** concorda com o mais próximo |
| Coletivo | singular: *A equipe **trabalha**.* |
| Partitivo (metade, a maioria de…) | singular ou plural: *A maioria dos alunos **passou/passaram**.* |
| Que / quem | concorda com o antecedente (*Fui eu que **fiz**.*) / quem → 3ª sing. ou conforme o ant. |
| Haver (= existir) | **impessoal**: *Há problemas.* (não “hão”) |
| Fazer (tempo) | impessoal: *Faz dois anos.* |
| Parecer + infinitivo | *Os arquivos **parecem** estar…* / *O arquivo **parece** estar…* |

### Verbos que confundem
- **Haver** (existir) ≠ **existir** (este varia: *Existem erros.*)
- **Ser** em horas/datas: *É uma hora.* / *São duas horas.*

## Concordância nominal

### Adjetivo com vários substantivos
- Depois: plural (*tecnologia e segurança **modernas***)  
  ou concorda com o mais próximo
- Antes: costuma concordar com o mais próximo

### Casos fixos (decore)

| Expressão | Forma |
|---|---|
| É **proibido** / **necessário** / **bom** + substantivo **sem** artigo | invariável: *É proibido entrada.* |
| Com artigo | varia: *É **proibida** a entrada.* |
| **Anexo**, **incluso**, **obrigado** | concordam: *Segue **anexa** a planilha.* / *Muito **obrigada**.* |
| **Menos**, **alerta**, **pseudo** | invariáveis |
| **Meio** (= um pouco) | invariável: *meio cansada* |
| **Meia** (= metade) | varia: *meia hora* |
| **Só** (= sozinho) | varia: *sós* |
| **Só** (= apenas) | invariável |

## Checklist

- [ ] “Haver” no sentido de existir → singular
- [ ] Núcleo do sujeito longe do verbo → não me engano
- [ ] Anexo/obrigado concordam; menos não
