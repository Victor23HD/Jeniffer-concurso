# Como gerar PDF (opcional)

Os materiais estão em **Markdown** (fáceis de ler no GitHub e no celular).

Para PDF:
1. Abra o arquivo `.md` no Cursor/VS Code ou no GitHub
2. Imprima a página (Ctrl+P)
3. Destino: **Salvar como PDF**

Ou use uma extensão “Markdown PDF” no VS Code.

Sugestão de PDFs para imprimir primeiro:
- `estudo/PLANO-DE-ESTUDOS.md`
- `estudo/informatica/` (todos — valem 20 questões)
- `estudo/flashcards/informatica.md`
- `simulados/simulado-01.md` + gabarito
