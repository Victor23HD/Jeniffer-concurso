# Flashcards

Use para revisão rápida (10–20 min/dia).

## Arquivos

| Arquivo | Conteúdo |
|---|---|
| `portugues.md` | Gramática e interpretação |
| `matematica.md` | Cálculos e fórmulas |
| `logica.md` | Raciocínio lógico |
| `informatica.md` | Específicos da prova |
| `*.csv` | Mesmo conteúdo (frente;verso) — dá para importar no Anki |

## Como estudar

1. Leia a pergunta  
2. Responda **em voz alta** ou no papel  
3. Confira o verso  
4. Errou? Marque e revise no dia seguinte  

No Anki: File → Import → escolha o CSV → campo separado por `;`
