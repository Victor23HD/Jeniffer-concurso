# Flashcards — Informática

**Q:** RAM × HD/SSD?  
**R:** RAM=volátil (trabalho); HD/SSD=armazena permanente.

**Q:** O que é driver?  
**R:** Software que permite o SO controlar o hardware.

**Q:** Hardware × software?  
**R:** físico × programas.

**Q:** Atalho no Windows é o arquivo?  
**R:** Não — apenas aponta para o arquivo/programa.

**Q:** Intranet × internet?  
**R:** intranet=rede interna; internet=rede pública mundial.

**Q:** Função do DNS?  
**R:** Traduz nome (URL) em endereço IP.

**Q:** Função do DHCP?  
**R:** Distribui IP automaticamente na rede.

**Q:** HTTPS × HTTP?  
**R:** HTTPS tem criptografia (mais seguro).

**Q:** O que é phishing?  
**R:** Golpe para roubar dados/senhas (site/e-mail falso).

**Q:** Malware é o mesmo que vírus?  
**R:** Malware é termo genérico; vírus é um tipo.

**Q:** Cco no e-mail serve para?  
**R:** Enviar cópia oculta (destinatários não se veem).

**Q:** Regra de backup 3-2-1?  
**R:** 3 cópias, 2 mídias, 1 off-site.

**Q:** Win+L faz o quê?  
**R:** Bloqueia o computador.

**Q:** No Excel, fórmula começa com?  
**R:** =

**Q:** Referência absoluta no Excel?  
**R:** $A$1 (trava coluna e linha)

**Q:** Animação × transição no PowerPoint?  
**R:** animação=objetos; transição=entre slides.

**Q:** Linux: pasta raiz?  
**R:** /

**Q:** Comando Linux para listar arquivos?  
**R:** ls

**Q:** Ransomware faz o quê?  
**R:** Criptografa dados e pede resgate.

**Q:** Switch × roteador?  
**R:** switch=liga dispositivos na LAN; roteador=liga redes/internet.

**Q:** MFA / 2FA?  
**R:** Autenticação com segundo fator além da senha.

**Q:** Princípios CIA da segurança?  
**R:** Confidencialidade, Integridade, Disponibilidade.
