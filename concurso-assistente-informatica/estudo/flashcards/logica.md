# Flashcards — Lógica

**Q:** Negação de “Todo A é B”?  
**R:** Algum A não é B.

**Q:** Negação de “Algum A é B”?  
**R:** Nenhum A é B.

**Q:** Negação de “Nenhum A é B”?  
**R:** Algum A é B.

**Q:** Quando “se P então Q” é falsa?  
**R:** Só quando P verdadeiro e Q falso.

**Q:** Afirmar o consequente é válido? (Se chove, molha. Molhou. Logo choveu.)  
**R:** Não — falácia.

**Q:** “E” lógico (P e Q) é verdadeiro quando?  
**R:** Só se ambos verdadeiros.

**Q:** “Ou” inclusivo é falso quando?  
**R:** Só se ambos falsos.

**Q:** Sequência: 2, 6, 12, 20, 30, ?  
**R:** 42 (diferenças +4,+6,+8,+10,+12)

**Q:** Em diagramas, “todos A são B” como desenhar?  
**R:** Círculo A dentro de B.

**Q:** “Alguns” significa?  
**R:** Pelo menos um.
