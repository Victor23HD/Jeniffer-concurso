## Comentários (breve)

**01.** "Porque" (junto, sem acento) introduz causa — conjunção causal equivalente a "pois".
**02.** Exceção, beneficente e privilégio seguem a ortografia oficial.
**03.** Verbo entregar + destinatário masculino: regência exige "ao diretor".
**04.** "Existem" concorda com "opções". Em A/B/C o verbo está errado; em E, "haver" (=existir) fica no singular: "Vai haver".
**05.** Ora subordinada adjetiva explicativa (que chegou cedo) vem entre vírgulas.
**06.** "Por que" separado: "por" (preposição) + "que" (pronome) — no sentido de "por que motivo".
**07.** "Dados" nomeia algo → substantivo (não verbo/advérbio/preposição/conjunção).
**08.** No início do período, sem palavra atrativa, usa-se ênclise: "Informaram-me".
**09.** Assistir (ver/prestar atenção) exige preposição "a"; antes de "apresentação" (fem.): "à".
**10.** O texto relaciona vulnerabilidade persistente ao comportamento dos usuários.
**11.** 12=2²×3; 18=2×3² → MMC=2²×3²=36.
**12.** 2400×1,15 = R$ 2.760,00.
**13.** Regra de três direta: 4→20; 6→x → x=30.
**14.** J=C×i×t=5000×0,02×3=R$300.
**15.** 3x=21 → x=7.
**16.** (8+12+16)/3 = 12.
**17.** h²=6²+8²=100 → h=10 cm (Teorema de Pitágoras).
**18.** PG de razão 2: 24×2=48.
**19.** Negação de universal afirmativa: existencial negativa.
**20.** Ambos os sistemas = interseção L ∩ W.
**21.** RAM é memória volátil para programas e dados em execução.
**22.** CPU executa instruções e processa dados.
**23.** Configurações do Windows centralizam ajustes do sistema.
**24.** Driver faz a ponte entre SO e dispositivo de hardware.
**25.** Incremental salva somente alterações desde backup anterior.
**26.** DNS resolve nomes (ex.: google.com) para IPs.
**27.** DHCP atribui IP, máscara, gateway e DNS aos clientes.
**28.** Cada octeto vai de 0 a 255; 192.168.1.1 é válido.
**29.** Intranet é rede interna, restrita à instituição.
**30.** Cco oculta endereços dos demais destinatários.
**31.** Phishing usa engenharia social para obter credenciais.
**32.** Ransomware sequestra dados mediante criptografia.
**33.** Referências relativas (A1) ajustam-se ao copiar.
**34.** $A$1 fixa linha e coluna (absoluta).
**35.** Ctrl+B aplica negrito (Bold).
**36.** Transição ocorre na mudança de slides; animação é dentro do slide.
**37.** Malware = software malicioso (vírus, trojan, etc.).
**38.** Suporte começa com diagnóstico: ouvir e verificar uso de CPU/RAM/disco.
**39.** Ubuntu é distribuição Linux de código aberto.
**40.** RAM: volátil e rápida. HD/SSD: armazenamento persistente.
