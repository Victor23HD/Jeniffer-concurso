## Comentários (breve)
**01.** CPU executa instruções e coordena operações do computador.
**02.** RAM é volátil — perde dados sem energia.
**03.** Gerenciador de Dispositivos controla hardware e drivers.
**04.** `ls` lista conteúdo de diretórios no Linux/Unix.
**05.** SSD usa flash, sem partes móveis, maior velocidade.
**06.** Firmware é software embarcado no hardware.
**07.** Incremental copia só o que mudou desde o backup anterior.
**08.** Restauração do Sistema reverte configurações a ponto de restauração.
**09.** BIOS/UEFI inicializa e testa hardware no boot.
**10.** Driver faz a ponte SO ↔ hardware.
**11.** Capacidade de memória mede-se em bytes (GB).
**12.** Root tem privilégios administrativos totais no Linux.
**13.** SO gerencia recursos e fornece interface ao usuário.
**14.** Pendrive armazena (E/S) e transporta dados.
**15.** Mirror duplica integralmente os dados.
**16.** CMD executa comandos em modo texto.
**17.** Placa-mãe integra CPU, memória e periféricos.
**18.** Quatro liberdades do software livre (FSF).
**19.** `chkdsk` verifica erros no sistema de arquivos/disco.
**20.** ISO é imagem de disco (CD/DVD/instalação).
